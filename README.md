## 项目初始化，必须执行(Necessary steps: initialization)
```
npm run init
```

## 本地运行(Run locally)
```
npm i 
npm start
```

## 本地部署 推荐使用pm2 (Pm2 is recommended for local deployment)
```
pm2 start ./bin/www
```

## 重置后台 (Reset the background)
```
npm run reset
```
